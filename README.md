# SimpleCashier

Hier ist mein Programm für eine einfache Kasse, welche benotet wurde.

- Preiseingabe
- Mehrwerststeuer(7% oder 19%)-Auswahl mit RadioButtons
- Addieren-Button
- Letztes Produkt wiederholen
- Totalausgabe
- Mwst 7% und 19% Ausgabe
- Netto-Ausgabe
- Zahlen-Button
- Bargeld-Feld
- Rückgeld-Ausgabe
- Nächster Bon Button
- Ein Kassenzettel mit Printfunktion
- Bei zuniedrigen Geldeingang, gibt er die Differenz aus
- ...
